####################### Fonctionnement du programme #######################
#______--------------------principale------------------______#


compilez à l'aide du Makefile

un dossier contenant des fichiers de tests est fourni sous le nom:
        f_test


un script bash permettant de faire une suite de tests des fichiers 
est fournis sous le nom de :
    testeur.sh    
    Ne pas oublier d'accorder la permission d'éxecution 
        du fichier avec la commande chmod +x testeur.sh

    vous pouvez lancer les tests en utilisant la commande:
    ./testeur.sh

sinon il est toujours possible de faire les tests unitaires
    avec ./main emplacement_du_fichier/fichier

Il y a en annexe un documents pdf expliquant plus en details le langage interprété
et sa grammaire

#______--------------------secondaire------------------______#

le fichier debug.c contient un entier binaire
si celui ci est vrai, les affichages pour le debugage seront activés

le fichier erreur_lecteur est un fichier qui permet de renvoyer les erreurs possibles

certaines modules du cours ont ete reutilisés

les fichiers sources qui commencent par test ont été utilisés lors 
de l'implementation du code pour tester certains modules
